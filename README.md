# 🍔 FoodBite - Responsive Restaurant Landing Page

## 📌 Overview

FoodBite is a modern and responsive restaurant landing page developed using HTML, CSS, and JavaScript. The project focuses on creating an attractive, user-friendly interface with a responsive layout that works seamlessly across desktop, tablet, and mobile devices.

This project was developed as **Task 1** of the **SkillCraft Technology Web Development Internship**.

---

## ✨ Features

- 🍽️ Modern restaurant landing page
- 📱 Fully responsive design
- 🧭 Smooth navigation
- 🎨 Clean and attractive UI
- ⚡ Fast loading interface
- 💻 Mobile-friendly layout
- 🌐 Cross-browser compatibility
